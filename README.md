# Focus Tracker

I made this because I noticed that while studying I get distracted a lot but I never actually track it.

This is a simple tool where I can start a study session and manually log when I get distracted.

## Features

- start and end study session
- track total time
- count distractions
- change status (studying / distracted)

## Tech Used

- HTML
- CSS
- JavaScript

## How it works

A timer runs using setInterval when study starts.

Distraction count increases manually when user clicks the button.

## Limitations

- distraction tracking is manual
- data is not saved
- UI is basic

## What I learned

- using timers in JS
- managing state using variables
- updating UI dynamically

## Why I made this

Mostly to understand how much I actually focus while studying.

It's simple but kind of useful.
